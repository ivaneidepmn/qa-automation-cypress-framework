# 🧪 QA Automation Cypress Framework

Professional QA Automation framework built with **Cypress**, applying **Page Object Model (POM)**, **BDD (Cucumber style)** and **GitLab CI/CD pipeline integration**.

---

## 🎯 Project Objective

This project demonstrates a scalable and professional test automation structure including:

- ✅ End-to-End testing
- ✅ Page Object Model (POM)
- ✅ Behavior Driven Development (BDD)
- ✅ Continuous Integration with GitLab CI
- ✅ Headless test execution
- ✅ Test artifacts (screenshots & videos)

---

## 🛠 Tech Stack

- Cypress
- JavaScript (ES6)
- Node.js
- GitLab CI/CD
- Chrome Headless
